Low Noise Amplifier (IBM 0.18μm CMRF7SF)

This project was developed using the Cadence environment and the IBM CMRF7SF design kit. It is a 2.4GHz Low Noise Amplifier (LNA)}, using a cascode common source amplifier configuration. The power comsuption is 37mW with 1.8V supply voltage (with buffer circuit for measurements). The gain is 19.7dB and noise figure is 2.55dB.
